About the Project

Problem Statement

In an increasingly energy-conscious world, understanding and predicting household energy consumption is crucial. This project addresses the need for a data-driven approach to smart energy management by leveraging machine learning to forecast usage patterns and identify key drivers of consumption.

Dataset Description

The analysis is based on the UK-DALE (UK Domestic Appliance-Level Electricity) dataset. This public dataset contains high-resolution energy consumption data from several homes in the UK.

  • Data Fields: Timestamp, aggregate energy usage (kWh), appliance-level usage (e.g., fridge, washing machine).
  • Preprocessing: The raw data was cleaned to handle missing values, resampled from per-second to hourly intervals, and enriched with time-based features like hour of the day, day of the week, and seasonality indicators.

Project Objective

The core objective is to develop and compare machine learning models that can accurately predict short-term, household-level energy consumption. This involves analyzing historical trends to identify key factors influencing usage and providing a tool for visualizing these patterns.
